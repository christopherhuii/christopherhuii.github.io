Blog #16: What is Rails?

10/17/2014
4:04 PM

This is a question I have been asking myself throughout Dev Bootcamp’s Phase 0. I know there is something known as ‘Ruby on Rails’, but I don’t know what ‘Rails’ is. During Unit 2 of Phase 0, we spend a lot of time learning about Ruby, but there was no sign of Rails. I still have not touched Rails, so I’m going to give it my best shot explaining Rails without any previous experience.

Rails is a software library that makes Ruby a more versatile language. Rails was created to extend the Ruby programming language. Rails is a framework that is used for building websites. Rails combines Ruby with HTML, CSS, and JavaScript in order to create functional web applications. By doing this, Rails makes it easier to collaborate and maintain web applications. Rails is great because it allows programmers to accomplish more while writing less code.
